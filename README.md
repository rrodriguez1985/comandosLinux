# Comandos Básicos de Linux
Listado de Comandos básicos utilizados para Sistemas Operativos Linux (Ubuntu, Kali Linux, Suse, CentOS, RHEL (Red Hat), Linux Mint...)
Es un resumen/chuletario básico para aquellos que están empezando a utilizar Linux y puedan conocer los comandos más comunes.
Especialmente útiles para alumnxs de SMR, ASIR, DAM y DAW en materias tales como:

- **Sistemas Operativos Monopuesto / Operativos en Red (SMR)**
- **Seguridad Informática (SMR)**
- **Implantación de Sistemas Operativos / Administarción de Sistemas Operativos (ASIR)**
- **Sistemas Informáticos (DAM /DAW)**
- **Ciberseguridad**
